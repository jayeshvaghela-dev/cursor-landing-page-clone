## About

This project is a **replica of the Cursor landing page**. The goal was to **recreate the overall look, feel, and structure**.  

- Colors closely match the original website.  

---

## Sections Implemented

The landing page includes the following sections:

1. **Top Navigation Bar** – Logo, navigation links, primary CTA, dark background.  
2. **Hero Section** – Main headline, description, CTA buttons, and large product screenshot.  
3. **Trusted By / Logos** – Row of company logos showcasing credibility.  
4. **Feature Sections** – Three blocks in two-column layout; alternating text/image positions.  
5. **Feature Cards Section** – Section title and grid of 3–4 cards highlighting features.  
6. **Testimonials** – Quote cards with customer name and role.  
7. **Use Cases / Stories** – Image cards with short descriptive text.  
8. **Changelog / Updates** – List of updates with dates.  
9. **Team / About** – Large image, short description, and CTA button.  
10. **Final CTA** – Prominent heading with single CTA button.  
11. **Footer** – Multi-column links, social icons, and company information.

## How to Run Locally

Follow these steps to run the Cursor Landing Page on your local machine:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/jayeshvaghela-dev/cursor-landing-page-clone.git
    ```

2. **Navigate into the project folder:**

    ```bash
    cd cursor-landing-page-clone
    ```

3. **Open the project:**

    - Open `index.html` in your browser, **or**  
    - Use a live server extension (like **Live Server** in VS Code) for real-time preview.
